DOM定位

DataTables 默认会打开部分特性，比如搜索框，分页显示等等，或许你不喜欢 DataTables 这样去布局，可能你想把分页按钮放在底部的中间，搜索框放在顶部的左上角，不用担心datatables考虑到这个问题，使用dom选项就可以灵活配置各个特性的位置

DataTables 定义了10个字符表示不同的组件

  • l - Length changing 每页显示多少条数据选项
  • f - Filtering input 搜索框
  • t - The Table 表格
  • i - Information 表格信息
  • p - Pagination 分页按钮
  • r - pRocessing 加载等待显示信息
  • < 和 > - div elements 一个div元素
  • <"#id" 和 > - div with an id 指定id的div元素
  • <"class" 和 > - div with a class 指定样式名的div元素
  • <"#id.class" 和 > - div with an id and class 指定id和样式的div元素

  • 示例2

相当于以下DOM结构:

<div class="wrapper">
    { filter }
    { length }
    { info }
    { paging }
    { table }
</div>
